# <a name="windows-virtual-desktop-on-surface"></a>Surface 上的 Windows 虚拟桌面

## <a name="introduction"></a>简介

WindowsSurface 上的虚拟桌面允许你在 Surface 设备上运行虚拟桌面基础结构 (VDI) — 模糊了本地桌面体验和虚拟桌面之间的界限，其中触摸、笔、墨迹和生物识别身份验证跨越物理和虚拟环境。 Surface 1 上的 Windows 虚拟桌面是计算演变中的另一个里程碑，它将 Microsoft 365（在 Azure 云中虚拟化）与 Surface 上 Windows 10 的高级安全保护、企业级可管理性和增强的生产力工具相结合。 <a href="#1"><sup> </sup></a> Azure 中高级外形因素和虚拟桌面基础结构的融合使组织能够提供高级用户体验、可移植性、安全性、业务连续性和新式管理。

### <a name="windows-virtual-desktop"></a>Windows 虚拟桌面

Windows虚拟桌面 (WVD) 是一项在 Azure 云中运行的综合桌面应用虚拟化服务。 它是唯一一个提供简化管理、多会话[Windows 10、Microsoft 365 应用版](https://www.microsoft.com/microsoft-365)优化和支持远程桌面服务 (RDS) 的虚拟桌面基础结构。 借助 WVD，可以快速部署和Windows Azure 上的桌面和应用，并获取内置的安全性和合规性功能。

### <a name="windows-virtual-desktop-partner-integrations"></a>Windows虚拟桌面合作伙伴集成

有关批准的合作伙伴提供商和独立软件供应商的列表，请访问 Azure [Marketplace](https://azuremarketplace.microsoft.com)并搜索Windows桌面。 某些合作伙伴还提供虚拟桌面即服务 (DaaS) 。 DaaS提供完全托管的交钥匙桌面和虚拟化服务， (虚拟机) 虚拟机与虚拟机之间保持连接。 通过向全球任何地方的用户提供自定义桌面的能力，公司可以快速调整以适应不断变化的市场情况，即根据需要在何时何地根据需要旋转云桌面。

- 灵活的**外形**设备 - 如 Surface Pro 7+、Surface Go 2、Surface Pro 7 和 Surface Pro X 等一体式设备（带笔、触摸和可分离键盘）。 你甚至可以使用 [Surface Duo](/surface-duo/) 连接到 WVD。

### <a name="virtual-gpus"></a>虚拟 GPU

GPU 非常适合 AI 计算和图形密集型工作负载，帮助客户通过高端远程可视化、深度学习以及预测分析等方案推动创新。 但是，这对需要远程工作或一直工作在外工作的专业人员来说并不理想，因为不同程度的内部 GPU 能力与物理设备绑定在一起，从而限制了移动性和灵活性。

为解决此问题，Azure 提供了 N 系列虚拟机系列，其 NVIDIA GPU 功能 (vGPU) 。 通过 vGPUs，IT 可以跨多个虚拟机共享 GPU 性能，或者通过向一个虚拟机分配多个 GPU 来为要求严格的工作负载提供电源。 对于 Surface，这意味着无论使用什么设备，从 Surface Go 2 到 Surface Laptop 4，你的设备都有权访问强大的服务器级图形性能。 Surface 和 vGPUs 允许你将你喜欢的 Surface 的所有功能组合在一起，包括笔、触摸、键盘、触控板和 PixelSense 显示器，并且图形功能仅在高性能计算环境中可用。

Azure N 系列将这些功能在 Surface 设备上实现，使你可以随时随地以任何想要的方式工作。 [详细了解 Azure N 系列和 GPU 优化的虚拟机大小。](/azure/virtual-machines/sizes-gpu)
